📦 Daten-Migration → Firebase

Diese Seite liest deine lokalen Daten aus dem alten Manager aus und lädt sie direkt in Firebase hoch — einmalig, dauert 2 Sekunden.

1
Diese Datei muss im gleichen Browser geöffnet werden, in dem du bisher die alte Manger.html benutzt hast (z.B. Edge oder Chrome)
2
Klick auf „Daten prüfen" — die gefundenen Daten werden angezeigt
3
Klick auf „Nach Firebase hochladen" — fertig!
Bereit. Klick auf „Daten prüfen" um zu starten.
